📸 MERN Photo Sharing App

A full-stack photo sharing web application built using the MERN stack. Users can upload images from their gallery, add captions, and share them seamlessly. Integrated ImageKit for optimized image storage and fast URL-based delivery.

🚀 Features
📤 Upload photos from gallery
📝 Add captions to posts
⚡ Fast image delivery using ImageKit
📱 Responsive and modern UI
🔄 Dynamic content rendering
🌐 Full-stack implementation (MongoDB, Express, React, Node.js)
🛠️ Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Image Handling: ImageKit
📂 Project Structure
client/     -> Frontend (React)
server/     -> Backend (Node + Express)
⚙️ Installation & Setup
Clone the repository
[git clone https://github.com/your-username/your-repo-name.git](https://github.com/Anshikasaxenaa/Post_app/new/main?filename=README.md)
cd Post_app
Install dependencies
cd client
npm install

cd ../server
npm install
Add Environment Variables
Create a .env file in the server folder and add:
MONGO_URI=your_mongodb_connection
IMAGEKIT_PUBLIC_KEY=your_key
IMAGEKIT_PRIVATE_KEY=your_key
IMAGEKIT_URL_ENDPOINT=your_url
Run the project
# Backend
cd server
npm start

# Frontend
cd client
npm start
